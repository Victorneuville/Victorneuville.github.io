HardSite

Description

HardSite est un projet que j’ai réalisé lors de mon année de terminale ayant choisi l’option ISN. Ce projet consistait à produire un site internet sur un sujet qui nous plaisait. Le projet devait être réalisé en groupe de 2, nous nous sommes donc mis d’accord pour créer un site qui regroupait des informations sur plusieurs styles de musiques et quelques DJs. Le projet a été réalisé en HTML/CSS et JavaScript.

Outils utilisés

  • HTML
  • CSS
  • Javascript

Travail realisé

Lors de ce projet, nous avons commencé par organiser le travail à réaliser. Ensuite, nous avons commencé à développer le site. Pour ma part, j’ai réalisé les contenus des différentes pages des DJs et j’ai réalisé le Template pour pouvoir y inscrire les contenus. J’ai également codé le mot de passe en JavaScript. Nous savions qu’il allait falloir demander les droits à tous les DJs, ce qui était impossible à notre échelle et il aurait fallu qu’ils acceptent tous. Le site n’a donc pas été publié.
