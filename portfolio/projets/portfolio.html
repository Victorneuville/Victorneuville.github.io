Portfolio

Description

Ce projet est un projet personnel dans lequel je me suis investi. Il consistait donc à réaliser un site web « from scratch » en HTML/CSS/JavaScript, qui serait un livrable en plus lors de mes recherches de job ou d’alternance. J’ai donc pensé au design de celui-ci pour qu’il me représente un maximum, tout en respectant les code UX et UI.

Outils utilisés

  • Figma
  • html
  • css
  • javascripte (jquery)
  • Photoshop
  • Illustrator

Travail realisé

Avant de commencer la réalisation de mon portfolio, je suis passé par plusieurs étapes, a commencé par la réalisation de ma charte graphique ainsi que mon logo. Ma charte graphique a été pensée pour les personnes atteintes d’un handicap, les couleurs, la police et les formes ont été réfléchis pour la lisibilité de tous. (AA validé, AAA partiellement). J’ai ensuite réalisé de nombreuses maquettes pour choisir celle qui me correspondait le mieux. Une fois toutes ces étapes réalisées, j’ai commencé l’intégration de mon site en HTML, CSS et JavaScript. Une fois le site développé entièrement, j’ai remplacé les images de substitution par les images ou les illustrations que j’ai réalisé sur Photoshop ou Illustrator. Enfin, j’ai hébergé mon portfolio sur github.
